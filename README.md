# AuditMySite Studio — Downloads

Official download repository for AuditMySite Studio.

> This repository contains compiled binary releases only.
> No source code is published here.

## Download

Go to [Releases](../../releases) and pick the installer for your platform.

| Platform | File |
|----------|------|
| Windows | `AuditMySite-Studio_x.y.z_x64-setup.exe` |
| macOS (Apple Silicon) | `AuditMySite-Studio_x.y.z_aarch64.dmg` |
| macOS (Intel) | `AuditMySite-Studio_x.y.z_x64.dmg` |
| Linux | `audit-my-site-studio_x.y.z_amd64.AppImage` |

## Verify

Each release includes a `checksums.txt` (SHA-256).

```bash
sha256sum -c checksums.txt
```

## Auto-Update

The app checks this repository's releases automatically for updates.
